# Hookmastery: Custom Hook for API Calls (alx-project-0x0C-setup)

his repository demonstrates the creation and usage of a custom hook for API calls in our image Generator application. Custom hooks are reusable functions that leverage React's state and lifecycle features.


## Getting Started

1. **Clone the repository:**

```bash
git clone https://github.com/yuslove1/alx-project-0x12-setup.git

```
2. **Install dependemcies:**

```bash
cd alx-project-0x12-setup
npm install

```

3. **Run the development server:**

```bash
npm run dev

```

Open http://localhost:3000 in your browser.

## Key Improvements

* Custom hook (useFetchData) for making API requests
* Simplified API call handling within components
